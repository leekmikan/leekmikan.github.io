Windows Media Player(Legacy)でコミケなどで買ったCDを取り込む
考えられる全パターン紹介

・数が少ない同人CDを取り込む際に起きる特有のトラブルの対処法を中心に記載

・基本1から順に進めていき、トラブルが発生した場合は分岐に従っていくと、最終的にゴールに到達します

(ボカロCDを300枚くらい取り込んだから、ほとんどのトラブルを網羅出来てるはず...)

以下、Windows Media PlayerをWMPと略す

1. CDをPCに挿入

PCに付いているCDドライブにCDを入れる

なければ外付けCDドライブを購入し、USB接続をしてください...

2. Windows Media Player(Legacy)を起動

検索欄から「Windows Media Player」と入力すると出てくると思うので、出てきたらクリックして開く

出てこなければ入ってないのでインストール(多分あり得ない?)

3. CDのマークがついたドライブを押す

WMP左側のCDマークがついたドライブを押す(多分、Eドライブ)

問題ない場合はこちら

違うCDの情報が表示される場合はこちら

情報が表示されない場合(曲名がトラック1,トラック2...のまま)はこちら

3-1. アルバムアートを再検索する

ここに来る原因は、WMPのデータベース(CCDB)が、「曲数と曲の長さ的にこれかな～?」というので別のアルバムと間違えられた場合

(例：プロセカの映画特典CD)

アルバムアート付近を右クリックし、「アルバム情報の検索」をクリック

該当のアルバムがあれば選択

問題ない場合はこちら

該当のアルバムが見つからない場合はこちら

3-2. WMPの設定を変更する

※2度ここに来てしまった人は、「4-1. ジャケット画像を用意」までネットワークを遮断してください。

WMP左上の「整理」から「オプション」を開き、「ライブラリ」を次の画像のような設定にする。

完了したら一旦CDを取り込み直す(「3-1-1. 手入力をする(めっちゃめんどい)」に移動するまでは同じ手順を踏む?)

3-1-1. 手入力をする(めっちゃめんどい)

ここに来る原因は、WMPのデータベース(CCDB)が、該当のアルバム情報を見つけられなかった場合

(例：超小規模の同人サークルのCD)

アルバムの各項目のテキストを、手元のCDを参考に手入力する

入力したいところを右クリックし、「編集」を押すと書き込める

(予測変換で)Tabキーを押すとうまく入力できないのでうっかり押さないこと。(めんどい..)

手入力箇所

  • アルバム
    • アルバム名
    • アーティスト名
    • ジャンル
    • 発売年
  • タイトル(曲名)
  • 参加アーティスト
  • 作曲者(参加アーティストと同じか空欄で良さそう?)

4. CDの取り込み

WMP上部にある「CDの取り込み」を押して取り込む

※画像は、取り込み完了状態

問題ない場合はこちら

ジャケット画像がない場合はこちら(取り込み後に実行)

4-1. ジャケット画像を用意

CDを取り外す

アルバム名でググるなどして画像を探す

見つけた画像を右クリックすると、「画像をコピー」のような文言が出てくるため選択

WMPに戻り、左側の「アルバム」を選択し、該当のアルバムを探す

右クリックすると、「アルバムアートの貼り付け」がでてくるので選択

問題ない場合はこちら

画像が見つからない場合はこちら

アルバムが2つ以上に分裂した場合はこちら

4-1-1. CDの取り込み直し

ここに来る原因は、CDの取り込み時に、何らかの問題が発生した場合。もう一度やり直すと成功する。

CDを取り外す

左側の「アルバム」を選択し、該当のアルバムを探す

右クリックすると、「削除」がでてくるので選択

コンピューター上からも削除する(たぶん　未確認)

※2度「3-2. WMPの設定を変更する」に来てしまった人は、再度「4-1. ジャケット画像を用意」までネットワークを遮断してください。

こちらに戻る

4-1-1-1. 画像をスキャン

プリンターを用意できない場合は飛ばしてください(こちらへ)

プリンターのスキャナーに、手元のCDのジャケットを読み込ませ、画像データに変換する

(解像度は結構小さくても良さそう...　300dpiでも大きかった)

後は、画像を正方形(大体で可)に切り取り、ctrl+Vでコピーする。

WMPに戻り、左側の「アルバム」を選択し、該当のアルバムを探す

右クリックすると、「アルバムアートの貼り付け」がでてくるので選択

こちらへ
