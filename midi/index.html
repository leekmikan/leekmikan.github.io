命令一覧

命令は太鼓さん次郎に近い形式

音階はカンマ、命令はセミコロンまたは改行区切りで入力　　　

例:

#HU 4;C4,E2,F5,-,


音階例

ド...C4　ファ♯...FS4　(低い)シ♭...BF3　ミのダブルシャープ...EDS4　ソ♭♭...GDF4　休符...-


↓#がつくシリーズ　x=自然数

#BPM x 　　BPM

#CH x 　　トラック(0～15　9はドラム専用)

#NUMERATOR x 　　 分子(テスト)

#DENOMINATOR x 　　 分母(テスト)

#HU x 　　 n分音符

#SF x 　　 調号 (-7～7)

#POWER x 　　 音量 (～127)

#TONE x 　　 音色 (～127)

#X x 　　 長さx倍 例)付点8分音符のラ #HU 16;#X 3;A4,#X 1; '#X 1で閉じる

#DELAY x 　　 x[ms]遅らせる
